# tesis-proyect
1- Descripcion
- Este proyecto es un sistema de Compras, Ventas y Servicios de Motocicletas , desarrolado para mi defensa de tesis. El proyecto se encuentra en proceso de desarrollo actualmente. 
Los modulos culminados actualmente son los refereneciales y en proceso de desarrollo actualmente el modulo de compra, funcionando Pedido y Orden de Compra en este ultimo.

2- Requisitos para la integracion.
- Tener instalado Servidor XAMPP server o WAMP server para correr el sistema el lado del servidor (particularmente utilizo XAMPP): 
links de descarga -XAMPP https://www.apachefriends.org/es/index.html

3- Pasos a seguir para la integracion
- Una vez instalado el servidor y clonado los archivos en su local area dentro de una carpeta, la carpeta respectivamente hay que moverla a la carpeta htdocs donde esta instalado
XAMPP. Ejemplo: C:\xampp\htdocs
- Una vez iniciado XAMPP hay que hacer correr los procesos de Apache y Mysql, una vez realizado, el servidor ya esta arriba
- La base de datos se encuentra en la carpeta bd, se debe abrir el administrador de BD desde el navegador http://localhost/phpmyadmin/, crear una nueva base de datos
llamado dbsistema, en dicha base, se debe importar los datos del archivo dbsistema.sql. Ejemplo como importar: https://www.youtube.com/watch?v=J_ow7VIQhxw&ab_channel=alb3Rth0o
- Obs: las dependecias ya se encuentran en la carpeta public

4- Tecnologias Utilizadas.
- PHP, Javascript, libreria Jquery, Bootstrap, Css (se utilizar una herramienta para el desarrollo de reportes, fpdp los mas probable).
